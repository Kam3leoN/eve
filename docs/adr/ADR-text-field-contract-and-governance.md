# ADR: text-field contract and governance

## Status
Accepted

## Context
Gouvernance composant `text-field` avec contrat typé, tests et quality gates CI.

## Decision
- Contrat machine-readable (text-field.contract.ts)
- Tests contract/a11y/visual
- Stabilité + migrations explicites
- Registre tokens composant

## Consequences
API plus stable, maintenance plus prédictible.
