# ADR: button-split contract and governance

## Status
Accepted

## Context
Gouvernance composant `button-split` avec contrat typé, tests et quality gates CI.

## Decision
- Contrat machine-readable (button-split.contract.ts)
- Tests contract/a11y/visual
- Stabilité + migrations explicites
- Registre tokens composant

## Consequences
API plus stable, maintenance plus prédictible.
