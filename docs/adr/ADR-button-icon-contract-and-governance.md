# ADR: button-icon contract and governance

## Status
Accepted

## Context
Gouvernance composant `button-icon` avec contrat typé, tests et quality gates CI.

## Decision
- Contrat machine-readable (button-icon.contract.ts)
- Tests contract/a11y/visual
- Stabilité + migrations explicites
- Registre tokens composant

## Consequences
API plus stable, maintenance plus prédictible.
