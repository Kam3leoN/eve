# ADR: loading-indicator contract and governance

## Status
Accepted

## Context
Gouvernance composant `loading-indicator` avec contrat typé, tests et quality gates CI.

## Decision
- Contrat machine-readable (loading-indicator.contract.ts)
- Tests contract/a11y/visual
- Stabilité + migrations explicites
- Registre tokens composant

## Consequences
API plus stable, maintenance plus prédictible.
