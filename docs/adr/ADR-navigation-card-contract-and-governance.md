# ADR: navigation-card contract and governance

## Status
Accepted

## Context
Gouvernance composant `navigation-card` avec contrat typé, tests et quality gates CI.

## Decision
- Contrat machine-readable (navigation-card.contract.ts)
- Tests contract/a11y/visual
- Stabilité + migrations explicites
- Registre tokens composant

## Consequences
API plus stable, maintenance plus prédictible.
