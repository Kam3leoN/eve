Eve

Ce dépôt contient la librairie (bundles dans dist/) et la documentation statique dans docs/.

Documentation locale : npm run dev à la racine (build la lib puis Vite sur la doc ; npm run docs:dev fait la même chose). Pour la page d’accueil statique sans doc : npm run dev:lib.

Prévisualiser le build doc : npm run docs:build puis npm run docs:preview.

Après déploiement GitHub Pages, la doc suit l’URL du dépôt (https://<user>.github.io/<repo>/).
